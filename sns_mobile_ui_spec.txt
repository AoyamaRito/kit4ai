SNS Mobile App UI Specification
===============================

Screen Size: 36x51 characters (mobile portrait)

Components:
1. Status Bar (time, battery)
2. Header (title, search, settings)
3. Post Feed (user posts with like/comment/share)
4. Stories Section
5. Bottom Navigation (5 tabs)

Layer Structure:
- スマホ外枠 (Z-Order: 1)
- ステータスバー (Z-Order: 2)
- ヘッダー (Z-Order: 3)
- 投稿1 (Z-Order: 4)
- 投稿2 (Z-Order: 5)
- 投稿3 (Z-Order: 6)
- ストーリーズ (Z-Order: 7)
- ボトムナビ (Z-Order: 8)
- テキスト (Z-Order: 100)
