Banking App UI Design Specification
====================================

Application: みずほ銀行モバイルアプリ
Screen Size: 80x41 characters

Features:
1. Header - Bank logo, user name, timestamp
2. Account Balance Card - Current balance with trend
3. Credit Card Info - Usage and available limit
4. Transaction History - Recent 10 transactions
5. Investment Portfolio - Stock allocation and performance
6. Asset Trend Chart - 6-month asset growth visualization
7. Quick Actions - Common banking operations
8. Security Footer - SSL connection status

Design Elements:
- Shadow effects for card-like appearance
- Progress bars for credit utilization
- Simple bar chart for asset trends
- Emoji icons for visual appeal
- Right-aligned amounts for easy reading

Security Features:
- Masked account numbers (*** prefix)
- Secure connection indicator
- User identification in header

Layer Structure:
- メインフレーム (Z-Order: 1)
- ヘッダー (Z-Order: 2)
- 残高カード (Z-Order: 3)
- クレジットカード (Z-Order: 4)
- 取引履歴 (Z-Order: 5)
- 投資ポートフォリオ (Z-Order: 6)
- 資産推移 (Z-Order: 7)
- クイックアクション (Z-Order: 8)
- フッター (Z-Order: 9)
- テキスト (Z-Order: 100)
