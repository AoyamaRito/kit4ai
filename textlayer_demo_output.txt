TextLayer Demo - Full-width Character Support
=============================================

Features:
- Dedicated text layer with full-width character support
- Frame layers remain simple (1 char = 1 grid)
- Text layer handles 2-char width calculation
- Centered and right-aligned text functions

'売上統計' -> 8 chars width
'ユーザー分析' -> 10 chars width
'月別売上推移グラフ' -> 18 chars width
'Hello World' -> 11 chars width
