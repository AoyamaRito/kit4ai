ASCII-Safe Hospital Management System UI
========================================

System: SAKURA GENERAL HOSPITAL EMR v2.1
Screen Size: 80x36 characters
Character Set: ASCII only (compatibility guaranteed)
Target Users: Medical staff (doctors, nurses, admin)

Design Principles:
- ASCII-only characters for universal compatibility
- 8-bit byte canvas for precise alignment
- Tabular data layout for medical information
- Function key shortcuts for efficiency
- Real-time status monitoring

Main Components:
1. Header - Hospital name, doctor, system version
2. Patient Information - ID, demographics, medical info
3. Daily Schedule - Appointment queue with status
4. Prescription Management - Drug list with dosages
5. Test Results - Latest lab values and vitals
6. System Status - Database and backup monitoring
7. Function Keys - Quick access to main operations

Compatibility Features:
- Works in any text editor or terminal
- No special fonts or Unicode required
- Printable on any ASCII-compatible printer
- Network-safe transmission
- Legacy system compatible
