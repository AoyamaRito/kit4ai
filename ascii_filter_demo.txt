ASCII Filter Demo
=================

Full-width characters are automatically filtered out
Only ASCII characters (0-127) are preserved

Test Results:
'Hello 世界 World!' -> 'Hello  World!'
'Test メッセージ Message' -> 'Test  Message'
'ASCII only: 123 ABC' -> 'ASCII only: 123 ABC'
'Mixed: こんにちは Hello!' -> 'Mixed:  Hello!'
'日本語のみ' -> ''
'English only' -> 'English only'

Canvas Layout:
+----------------------------------------------------------+
|                                                          |
| Hello  World!                                            |
|                                                          |
| Test  Message                                            |
|                                                          |
| ASCII only: 123 ABC                                      |
|                                                          |
| Mixed:  Hello!                                           |
|                                                          |
+----------------------------------------------------------+
