POS Terminal System UI Design Specification
===========================================

System: KONBINI STORE POS SYSTEM v3.2
Terminal Type: Retail Point of Sale
Screen Size: 80x41 characters
Target Users: Store cashiers and managers

Core Functions:
1. Product Scanning and Entry
   - Barcode scanning support
   - Manual product code entry
   - Quantity adjustment
   - Price lookup and calculation

2. Transaction Management
   - Real-time subtotal calculation
   - Automatic tax calculation (10%)
   - Discount and promotion handling
   - Customer loyalty points integration

3. Payment Processing
   - Multiple payment methods support
     * Cash payments
     * Credit/Debit cards
     * IC cards (contactless)
     * QR code payments
     * Loyalty points redemption
   - Card authorization processing
   - Receipt printing control

4. System Integration
   - Inventory management sync
   - Sales reporting
   - Network connectivity monitoring
   - Hardware status tracking

Hardware Components:
- Barcode scanner
- Card reader (magnetic stripe/chip/contactless)
- Receipt printer
- Cash drawer
- Customer display
- Network connection

Function Key Operations:
F1 - Add Item (manual entry)
F2 - Delete Item (remove from transaction)
F3 - Apply Discount
F4 - Points Management
F5 - Payment Processing
F6 - Cancel Transaction
F8 - Manager Functions
ESC - Logout/Exit

Security Features:
- Cashier authentication
- Manager authorization for special functions
- Transaction logging
- Secure payment processing
- Data backup scheduling
