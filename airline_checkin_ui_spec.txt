Airline Check-in Kiosk UI Design Specification
===============================================

Application: スカイライン航空 セルフチェックインキオスク
Screen Size: 80x46 characters
Target Users: 航空会社の乗客（セルフサービス）

Features:
1. Header - Airline branding, countdown timer, current time
2. Flight Information - Flight details with aircraft icon
3. Boarding Pass Preview - Real-time boarding pass display
4. Seat Selection - Interactive seat map with availability
5. Action Menu - Check-in completion and additional services
6. Progress Status - Step-by-step completion tracking
7. Footer - Help, language options, cancel

UI Components:
- Rounded boxes for modern card-like appearance
- Seat map with visual seat availability (■□❌)
- Progress bar for completion tracking
- QR code placeholder for boarding pass
- Emoji icons for intuitive navigation

Accessibility Features:
- Large, clear text for easy reading
- High contrast visual elements
- Multilingual support option
- Touch-friendly button spacing

Business Logic:
- Real-time flight information display
- Seat availability checking
- Boarding pass generation
- Email/print options for boarding pass
- Additional service selection (meals, baggage)

Layer Structure:
- メインフレーム (Z-Order: 1)
- ヘッダー (Z-Order: 2)
- フライト情報 (Z-Order: 3)
- 搭乗券 (Z-Order: 4)
- 座席選択 (Z-Order: 5)
- ステータス (Z-Order: 6)
- アクション (Z-Order: 7)
- フッター (Z-Order: 8)
- テキスト (Z-Order: 100)
